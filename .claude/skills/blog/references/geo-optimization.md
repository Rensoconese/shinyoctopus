# GEO/AEO Optimization — Reference

## What LLMs Need to Cite You

LLMs cite content that is:
1. **Factually dense** — stats with sources every 150-200 words (+33.9% visibility)
2. **Structurally extractable** — self-contained paragraphs of 134-167 words (4.2x more selection for AI Overviews)
3. **Question-aligned** — H2s match conversational queries
4. **Table-rich** — HTML tables get +42% citations in Perplexity
5. **Fresh** — content updated in last 30 days gets 3.2x more AI citations
6. **Schema-marked** — pages with schema have 36% more probability of appearing in AI summaries

## Platform-Specific Patterns

### ChatGPT (87.4% of AI referral traffic)
- Cites from training data (static) + Bing index (with Search enabled)
- Wikipedia is source #1 (43% of general queries)
- Correlation with Google rankings: only 14%
- Brand search volume correlates 0.334 with citations
- Present on 4+ platforms = 2.8x more likely to be cited

### Perplexity
- Real-time index, 91% of citations come from Google top-10
- Reddit is 46.7% of its top-10 sources
- Highest sensitivity to freshness — decay at 2-3 days
- Tables get +42% boost vs prose
- API vs UI produce DIFFERENT citation results — UI cites more aggressively

### Google AI Overviews (25.8% of US searches)
- 52% of sources are Google top-10
- Multi-modal content (text + images + video + schema) = +317% selection
- E-E-A-T signals correlation r=0.81
- 40-60% of sources change month to month (high volatility = opportunity)

### Claude.ai
- Uses **Brave Search** as backend — content must be indexed in Brave
- Claude-SearchBot (allow) vs ClaudeBot (training, can block)
- Values: expertise depth, structured content, entity definition, recency

## GEO Audit Checklist

For each post, verify:

- [ ] First paragraph after each H2 is an Answer Capsule (40-60 words, self-contained)
- [ ] Every H2 is a conversational question
- [ ] At least 3 bolded statistics with year + source
- [ ] At least 1 comparison table
- [ ] FAQ section with 4-6 questions
- [ ] Internal links to service pages
- [ ] `dateModified` / `updatedDate` is current
- [ ] Content visible without JavaScript (Astro SSG = automatic pass)
- [ ] Schema JSON-LD in `<head>` (Article schema auto-generated by blog template)

## Citation Capsule Pattern

When writing a key fact that you want LLMs to cite, use this pattern:

```
**[Statistic]** [context that makes it self-contained] ([Source, Year]).
```

Example:
"**94.8% of the top one million homepages have detectable WCAG failures**, with an average of **51 accessibility errors per page** (WebAIM 2025)."

This is extractable, attributed, and self-contained — an LLM can cite it without surrounding context.

## Content Freshness Protocol

Update posts at these intervals:
- **30 days**: Check if statistics are still current. Update `updatedDate`.
- **90 days**: Refresh opening paragraph with current year. Add new data if available. Update stats.
- **180 days**: Full review. Replace outdated sections. Add new FAQ questions. Update all dates.

Posts not updated for 90+ days lose Perplexity citations significantly.
