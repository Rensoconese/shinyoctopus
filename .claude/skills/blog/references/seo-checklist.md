# SEO Checklist — Technical Validation

## Pre-Publish Checks

Run these checks on every post before publishing:

### 1. Title Tag
- [ ] Under 60 characters (Google truncates at ~60)
- [ ] Contains primary keyword
- [ ] Unique — no other page on the site has the same title
- [ ] Format: "Post Title | Shiny Octopus" (auto-added by blog template)

### 2. Meta Description
- [ ] 120-155 characters
- [ ] Contains primary keyword naturally
- [ ] Is a value proposition, not a summary
- [ ] Unique per page

### 3. Heading Hierarchy
- [ ] H1 exists only in frontmatter title (rendered by template)
- [ ] H2s follow logically from the topic
- [ ] No skipped levels (H2 → H4)
- [ ] All H2s are conversational questions

### 4. URL / Slug
- [ ] Slug is short and keyword-rich
- [ ] Uses hyphens, not underscores
- [ ] No stop words unless needed for readability
- [ ] File name = slug: `src/content/blog/[slug].md`

### 5. Internal Links
- [ ] At least 2 internal links to other pages
- [ ] Links to relevant service page (/seo, /accessibility, /servers)
- [ ] Descriptive anchor text (not "click here")
- [ ] No broken internal links

### 6. External Links
- [ ] Stats linked to authoritative sources
- [ ] Links open correctly (markdown links work in Astro)
- [ ] No links to competitors

### 7. Images
- [ ] `heroImage` set in frontmatter (if applicable)
- [ ] All images have alt text in markdown: `![descriptive alt](path)`
- [ ] Images are optimized (use Astro Image component for processing)

### 8. Schema (Auto-Generated)
The blog template at `src/pages/blog/[slug].astro` auto-generates:
- Article schema with headline, description, datePublished, dateModified, wordCount, author, publisher, keywords
- BreadcrumbList schema (Home > Blog > Post Title)
- Organization schema (global from Layout)

Verify after build: `grep 'application/ld+json' dist/blog/[slug]/index.html`

### 9. Open Graph
- [ ] og:type should be "article" for blog posts (NOTE: currently set to "website" — known issue)
- [ ] og:image set (falls back to site default)
- [ ] Title and description propagate correctly

### 10. Canonical URL
- [ ] Auto-generated by Layout from `Astro.url.href`
- [ ] Verify trailing slash consistency (site uses trailing slashes)

## Post-Publish Checks

- [ ] Post appears in sitemap (`dist/sitemap-0.xml`)
- [ ] Post has `<lastmod>` in sitemap
- [ ] Post renders correctly at the live URL
- [ ] JSON-LD schema validates at https://validator.schema.org
- [ ] Rich results test passes at https://search.google.com/test/rich-results
